Cloud

Cloud Native is the name of the game, with major players like Amazon Web Services (AWS), Microsoft Azure, and Google Cloud Platform (GCP). However, it's not solely focused on cloud infrastructure. Orchestration services like Kubernetes (K8s), Docker, and Terraform have further abstracted the cloud.

DevOps

DevOps is the integration of software development and IT operations. It employs practices and tools to enhance efficiency and speed up the development life cycle. By adopting DevOps, teams can continuously integrate and deliver their work using tools like Jenkins, Argo, Travis CI, ArgoCD, FluxCD, and Helm, known as CI/CD.
